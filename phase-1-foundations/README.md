# Phase 1: Kubernetes and IaC Foundations

Goal

- Become dangerous with Kubernetes mechanics and Terraform/GCP so you can provision and operate clusters confidently.

Modules (recommended order)

- [Module 1.1: GKE and Kubernetes Mechanics](module-1-1-kubernetes-gke.md)
- [Module 1.2: Terraform and GCP Fundamentals](module-1-2-terraform-gcp.md)

Completion criteria

- You can run and debug Kubernetes workloads in a local or GKE cluster.
- You can provision, modify, and destroy a GKE cluster via Terraform.
- You can reason about K8s scheduling and reconcile failures.
